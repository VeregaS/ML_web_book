# ML Web Book — Образовательная платформа по Машинному Обучению

Это интерактивное веб-приложение для изучения машинного обучения. Платформа объединяет строгую математическую теорию, интерактивные визуализации и полноценную среду выполнения Python (на базе Pyodide) прямо в браузере.

---

## 🤖 РУКОВОДСТВО ДЛЯ ИИ (NotebookLM / LLM)

**Внимание ИИ-ассистенту:** Если вы читаете этот файл, ваша задача — выступать в роли методиста, архитектора учебных программ и эксперта по ML. Пользователь предоставит вам базу знаний (книги, лекции). Ваша цель — генерировать контент в строгом соответствии с иерархией: **Модули -> Главы -> Уроки**.

Ниже описаны технические возможности платформы и форматы данных.

### 1. Иерархия контента
1.  **Модуль (Module):** Крупный блок обучения (например, "Нейронные сети"). Отображается в виде большой плитки на главной странице.
2.  **Глава (Chapter):** Тематический раздел внутри модуля (например, "Функции активации"). Группирует уроки.
3.  **Урок (Lesson):** Конкретная тема с теорией и практикой.

---

### 2. Возможности платформы
* **Теория:** Полный Markdown (включая таблицы).
* **Математика:** LaTeX через KaTeX. Используйте `$...$` для строчных формул и `$$...$$` для блочных.
* **Глоссарий (Умные подсказки):** Любое слово в теории, совпадающее с ключом из объекта `GLOBAL_GLOSSARY`, становится кликабельным. **Вы обязаны выделять ключевые термины жирным шрифтом.**
* **Визуализации (ChartBlock):** Встраиваемые SVG графики через блок кода `chart`.
* **Код (Python):** Выполнение кода в браузере. Проверка через `unittest`.

---

### 3. Формат Глоссария (`src/utils/glossary.js`)
```javascript
export const GLOBAL_GLOSSARY = {
  'Название термина': {
    definition: `**Название** — определение с формулами $$...$$`,
    example: `Аналогия или код \`\`\`python ... \`\`\``
  }
};
```

---

### 4. Формат Уроков (`src/content/lessons.js`)
**ВАЖНО:** Каждый урок должен иметь поля `module` и `chapter`.

```javascript
export const lessons = [
  {
    id: "unique-id",
    module: "Название Модуля", // Должно быть одинаковым для группы уроков
    chapter: "Название Главы",  // Группирует уроки внутри модуля
    title: "Урок X: Название",
    difficulty: "Легко" | "Средне" | "Сложно",
    description: "Краткое описание.",
    steps: [
      {
        id: "step-1",
        theory: "Текст с **Жирными Терминами** и ```chart ... ```",
        task: { 
          type: "code", // или "quiz"
          content: "..." 
        },
        initialCode: "...",
        testCode: "import unittest..."
      }
    ]
  }
];
```
